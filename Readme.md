[迷宫游戏](https://maze.hancel.org)是一款摸鱼游戏。这是 [摸鱼派客户端](https://github.com/imlinhanchao/fishpi-desktop) 迷宫游戏扩展。

## 安装方式

1. 下载 [Release](https://github.com/imlinhanchao/fishpi-ext-maze/releases) 最新版本的压缩包。
2. 解压缩到客户端扩展目录。
3. 重新开启客户端即可。

## 功能说明
扩展会读取登录用户信息，用于作为迷宫游戏的登录账号。
